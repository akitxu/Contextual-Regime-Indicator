# Confidence and Risk Index (CRI) — Documentation Hub

Bienvenido a la documentación oficial del **Confidence and Risk Index (CRI)**, un indicador cuantitativo diseñado para evaluar la salud actual de un activo financiero mediante la combinación de estabilidad (Confidence) y riesgo (Risk).

Este sitio reúne toda la información técnica, conceptual y práctica del CRI V7‑3.
